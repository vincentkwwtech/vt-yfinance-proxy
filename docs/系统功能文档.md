# VT YFinance Proxy 系统功能文档

## 1. 系统概述

VT YFinance Proxy 是一个基于 FastAPI 构建的 Yahoo Finance 数据代理 API 服务，用于内部网络环境下获取金融市场数据。系统封装了 yfinance 开源库，提供标准化的 RESTful API 接口，支持股票行情、财务数据、期权链、分析师评级等多维度金融数据查询。

**系统版本：** 0.1.0

## 2. 功能模块总览

| 模块 | 路径前缀 | 说明 |
|------|---------|------|
| 健康检查 | `/health` | 服务健康状态探测 |
| 金融数据 API（v2） | `/finance-data/yfinance` | 8 个金融数据查询端点 |
| 行情 API（v1/legacy） | `/api/v1` | 3 个基础行情查询端点 |

## 3. 功能详细说明

### 3.1 健康检查

| 项目 | 说明 |
|------|------|
| 端点 | `GET /health` |
| 功能 | 检测服务是否正常运行 |
| 返回示例 | `{"status": "ok"}` |

### 3.2 金融数据 API（v2）

所有端点均位于 `/finance-data/yfinance` 路径下，通过 `ticker` 查询参数指定股票代码。

#### 3.2.1 公司概况（Profile）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/profile?ticker={symbol}` |
| 功能 | 获取公司基本信息 |

**返回字段：**

| 字段 | 说明 |
|------|------|
| ticker | 股票代码 |
| name / longName | 公司名称 / 全称 |
| industry | 所属行业 |
| sector | 所属板块 |
| marketCap | 市值 |
| enterpriseValue | 企业价值 |
| currency | 货币单位 |
| exchange | 交易所 |
| quoteType | 报价类型 |
| website | 公司官网 |
| longBusinessSummary | 公司业务简介 |
| fullTimeEmployees | 全职员工数 |
| address / city / state / zip / country / phone | 公司地址信息 |

#### 3.2.2 股东信息（Shareholders）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/shareholders?ticker={symbol}` |
| 功能 | 获取股东持仓数据 |

**返回字段：**

| 字段 | 说明 |
|------|------|
| major_holders | 主要股东 |
| institutional_holders | 机构股东 |
| insider_transactions | 内部人交易记录 |
| mutualfund_holders | 共同基金持仓 |

#### 3.2.3 公司行为（Actions）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/actions?ticker={symbol}` |
| 功能 | 获取分红与拆股等公司行为记录 |

**返回字段：**

| 字段 | 说明 |
|------|------|
| actions | 合并的公司行为记录 |
| dividends | 分红历史 |
| splits | 拆股历史 |

#### 3.2.4 分析师评级（Analyst）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/analyst?ticker={symbol}` |
| 功能 | 获取分析师评级与目标价 |

**返回字段：**

| 字段 | 说明 |
|------|------|
| upgrades_downgrades | 评级变动记录 |
| recommendations | 推荐记录 |
| target_mean_price | 目标均价 |
| target_high_price | 目标最高价 |
| target_low_price | 目标最低价 |
| target_median_price | 目标中位价 |
| recommendation_key | 推荐关键字（如 buy/hold/sell） |
| recommendation_mean | 推荐均值 |
| number_of_analyst_opinions | 分析师数量 |

#### 3.2.5 财务数据（Financial）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/financial?ticker={symbol}` |
| 功能 | 获取完整财务报表与关键指标 |

**返回字段：**

| 字段 | 说明 |
|------|------|
| income_stmt | 年度利润表 |
| quarterly_income_stmt | 季度利润表 |
| balance_sheet | 年度资产负债表 |
| quarterly_balance_sheet | 季度资产负债表 |
| cashflow | 年度现金流量表 |
| quarterly_cashflow | 季度现金流量表 |
| key_metrics | 关键财务指标 |

**关键财务指标（key_metrics）包括：**

- trailingPE / forwardPE — 市盈率
- priceToBook — 市净率
- returnOnEquity / returnOnAssets — ROE / ROA
- debtToEquity — 资产负债率
- profitMargins / operatingMargins / grossMargins — 利润率指标
- revenueGrowth / earningsGrowth — 增长指标
- totalDebt / totalCash — 债务/现金头寸
- freeCashflow / operatingCashflow — 现金流指标

#### 3.2.6 期权链（Option Chain）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/optionchain?ticker={symbol}&expiration={date}` |
| 功能 | 获取期权到期日列表及指定日期的看涨/看跌期权数据 |

**参数：**

| 参数 | 必填 | 说明 |
|------|------|------|
| ticker | 是 | 股票代码 |
| expiration | 否 | 期权到期日（如不传则仅返回可选到期日列表） |

**返回字段：**

| 字段 | 说明 |
|------|------|
| expirations | 可选到期日列表 |
| calls | 看涨期权数据（仅指定 expiration 时返回） |
| puts | 看跌期权数据（仅指定 expiration 时返回） |

#### 3.2.7 K线数据（Kline）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/kline?ticker={symbol}&period={period}&interval={interval}&prepost={bool}` |
| 功能 | 获取历史 OHLCV K线数据 |

**参数：**

| 参数 | 必填 | 默认值 | 可选值 |
|------|------|--------|--------|
| ticker | 是 | — | 股票代码 |
| period | 否 | 1y | 1d, 5d, 1mo, 3mo, 6mo, 1y, 2y, 5y, 10y, ytd, max |
| interval | 否 | 1d | 1m, 2m, 5m, 15m, 30m, 60m, 90m, 1h, 1d, 5d, 1wk, 1mo, 3mo |
| prepost | 否 | false | true / false（是否包含盘前盘后数据） |

**返回字段：**

| 字段 | 说明 |
|------|------|
| ticker | 股票代码 |
| period | 请求周期 |
| interval | K线间隔 |
| prepost | 是否含盘前盘后 |
| data | OHLCV 数据数组 |

#### 3.2.8 全量数据（All）

| 项目 | 说明 |
|------|------|
| 端点 | `GET /finance-data/yfinance/all?ticker={symbol}` |
| 功能 | 一次性获取以上所有数据的聚合结果 |

**返回字段：** 包含 profile、shareholders、actions、analyst、financial、optionchain、kline 七个子对象。

### 3.3 行情 API（v1/Legacy）

#### 3.3.1 股票日线

| 项目 | 说明 |
|------|------|
| 端点 | `GET /api/v1/stocks/{symbol}/daily?start={date}&end={date}` |
| 功能 | 获取指定股票的日线 OHLCV 数据 |
| 参数 | symbol（路径参数）、start/end（可选，YYYY-MM-DD 格式） |

#### 3.3.2 VIX 日线

| 项目 | 说明 |
|------|------|
| 端点 | `GET /api/v1/vix/daily?start={date}&end={date}` |
| 功能 | 获取 VIX 恐慌指数的日线数据 |
| 参数 | start/end（可选，YYYY-MM-DD 格式） |

#### 3.3.3 实时报价

| 项目 | 说明 |
|------|------|
| 端点 | `GET /api/v1/stocks/{symbol}/quote` |
| 功能 | 获取指定股票的最新报价信息 |

**返回字段：** symbol、price、previous_close、open、day_high、day_low、volume、currency、exchange、market_time

## 4. 错误处理

| HTTP 状态码 | 场景 | 说明 |
|-------------|------|------|
| 200 | 正常返回 | 请求成功 |
| 404 | 无效股票代码 | InvalidTickerError |
| 422 | 参数校验失败 | 缺少必填参数或参数格式错误 |
| 500 | 数据获取异常 | FinanceDataError / YFinanceError |
